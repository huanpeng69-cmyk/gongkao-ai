## 如果还有问题

### 问题：编译时仍然提示中文引号错误

**原因**：浏览器或Next.js缓存了旧文件

**解决方案：**

1. **完全停止开发服务器**（Ctrl+C）

2. **手动删除 `.next` 文件夹**
   - 在Windows文件资源管理器中打开项目文件夹
   - 找到 `.next` 文件夹
   - 右键 → 删除（如果提示权限问题，关闭所有相关程序后重试）

3. **清理浏览器缓存**
   - Chrome: Ctrl+Shift+Delete → 清除缓存
   - 或使用无痕模式：Ctrl+Shift+N

4. **重新启动**
   ```bash
   npm run dev
   ```

### 问题：图片仍然显示慢

**检查项：**

1. **网络连接**：图片来自外部URL，需要稳定网络
2. **浏览器控制台**：按F12查看是否有图片加载错误
3. **图片URL**：确保图片URL可访问

**进一步优化建议：**
- 使用CDN加速图片
- 服务端缓存图片
- 使用图片代理服务

## 技术细节

### 图片优化原理

1. **懒加载**：只加载可见区域的图片
2. **预加载**：切换题目前提前加载下一题图片
3. **缓存**：已加载图片存入内存，避免重复请求
4. **重试**：加载失败自动重试3次
5. **占位**：加载中显示shimmer动画，避免布局抖动

### 新增工具模块

`lib/image-optimizer.ts` 提供：
- `preloadImage(src)` - 单图预加载
- `preloadImages(sources)` - 批量预加载
- `clearImageCache()` - 清理缓存
